#  Gesture-Controlled Neon Music Visualizer (Python)

A real-time, **gesture-controlled neon visualizer** built using **Python, OpenCV, and MediaPipe**.  
Your hand becomes the controller — visuals react instantly to gestures like **open palm, swipe, and pinch**.

No keyboard.  
No mouse.  
Just gestures. ✋✨

---

##  Features

- ✋ **Open Palm** → Increase visual energy  
- 👉 **Swipe Left / Right** → Switch visual modes  
- 🤏 **Pinch** → Change neon color / trigger effects  
-  Multiple visual modes:
  - Bars
  - Circle
  - Wave
  - Tunnel (fake 3D depth illusion)
-  Neon glow and particle effects
-  Smooth real-time performance

---

## Tech Stack
- Python 3.10+
- OpenCV
- MediaPipe (Hands Landmarker – Tasks API)
- NumPy
